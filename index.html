HTML

HTML stands for Hypertext Markup Language. This is a tag based language where tags used are created with greater than and less than signs.

For Example

<p> is a paragraph </p>

There are special characters such as &lt; would be a less than sign.

Inline VS Block Elements

Inline means that it is inline with the text. Block level forces a new line

Block Level: div, p, h1, h2, h3, h4, h5, h6, li

Inline elements are span, img, link, a, strong, em
